# 国内镜像配置说明（VoiceBillRecord 开发机）

已在当前用户环境做如下配置（便于 `brew install`、`npm install` 走国内线路）。**以下为个人本机配置摘要**，换电脑需重做或复制相同片段。

## 1. Homebrew（清华大学镜像）

写入位置：`~/.zshrc`（重新打开终端或 `source ~/.zshrc` 后生效）

| 变量 | 作用 |
|------|------|
| `HOMEBREW_API_DOMAIN` | `brew` 4.x 默认走 API 拉 formula 元数据，指向清华 |
| `HOMEBREW_BOTTLE_DOMAIN` | 预编译包（bottles）下载走清华，减轻 `ghcr.io` 直连压力 |
| `HOMEBREW_BREW_GIT_REMOTE` | `brew` 本体 `git pull` 上游 |
| `HOMEBREW_CORE_GIT_REMOTE` | `homebrew-core` 上游（与 `brew tap --custom-remote` 二选一场景见清华文档） |

官方说明：

- [Homebrew 镜像](https://mirrors.tuna.tsinghua.edu.cn/help/homebrew/)
- [Homebrew Bottles](https://mirrors.tuna.tsinghua.edu.cn/help/homebrew-bottles/)

配置生效后建议执行：

```bash
brew update
```

若曾卡住，可重新执行 `brew install node`。

## 2. npm（npmmirror，原淘宝镜像，由阿里维护）

写入位置：`~/.npmrc`

```text
registry=https://registry.npmmirror.com
```

验证：

```bash
npm config get registry
```

官方站点：<https://npmmirror.com/>

## 3. 复原官方源（需要时）

- **npm**：编辑 `~/.npmrc` 删除 `registry=` 一行，或执行  
  `npm config delete registry`
- **Homebrew**：删除 `~/.zshrc` 中「国内镜像」整段注释块后，按清华帮助页 [「复原仓库上游」](https://mirrors.tuna.tsinghua.edu.cn/help/homebrew/) 执行 `unset` 与 `git remote set-url`。

## 4. 与项目 `.env` 的关系

镜像只影响 **包下载**（brew、npm），与仓库里的 **`.env` / Neon / 大模型密钥`** 无关。
